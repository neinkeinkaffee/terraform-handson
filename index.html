Terraform

  • Declarative configuration language by HashiCorp
  • Helps specify cloud infrastructure as code
  • AWS cli calls under the hood (cloud API)

Resources

  • Specify (almost) anything as a resource
  • Combination of many resources may be required to get a thing done (examples basic network, service)

Stacks

  • All tf-files within a directory are a stack
  • Terraform keeps track of changes to a stack in a state file
  • Can import resources created in other stacks as data sources (not to be abused)
